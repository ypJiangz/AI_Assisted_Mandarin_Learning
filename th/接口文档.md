# 接口文档

## 定义

1. 返回数据格式

| 返回字段 | 字段类型 | 说明                                          |
| -------- | -------- | --------------------------------------------- |
| code     | int      | 返回结果状态。0：正常；其他见接口错误码中定义 |
| message  | string   | 错误码对应的说明                              |
| result   | {}       | 数据内容                                      |

2. 接口错误码

| 错误码 | 说明                                    |
| ------ | --------------------------------------- |
| 200    | 调用成功                                |
| 401    | 未知错误,该错误表达第三方系统出现的问题 |
| 4XX    | 客户端错误                              |
| 5XX    | 服务器错误                              |
